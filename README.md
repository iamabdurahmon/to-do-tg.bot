# To Do List Application

A clean, minimal, and efficient web-based task management application that helps you organize and track your daily tasks with ease.

## Features

- **Add Tasks**: Quickly add new tasks to your to-do list with a simple input field
- **Complete Tasks**: Mark tasks as completed with a checkbox - completed tasks are visually struck through
- **Edit Tasks**: Modify existing tasks by clicking the edit button
- **Delete Tasks**: Remove tasks from your list with the delete button
- **Task Counter**: Keep track of remaining pending tasks at a glance
- **Persistent Storage**: All tasks are automatically saved to your browser's local storage, so your list persists even after closing the browser
- **Motivational Quote**: Get inspired with a daily motivational quote below your task list
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Technologies Used

- **HTML**: Structure and semantic markup
- **CSS**: Modern styling with custom properties and responsive layout
- **JavaScript**: Dynamic functionality and task management logic
- **Local Storage API**: Persistent data storage without a backend
- **Remix Icon**: Beautiful icon library for UI elements

## How to Use

1. Open `index.html` in your web browser
2. Type your task in the input field
3. Click the add button (or press Enter) to add a task
4. Check the checkbox to mark a task as completed
5. Use the edit button to modify a task
6. Use the delete button to remove a task

## Local Storage

Your tasks are automatically saved in your browser's local storage under the key `todo-list`. This means:

- Your tasks are saved locally on your device
- No account or login required
- Data persists across browser sessions
- Clearing browser cache will delete your tasks
